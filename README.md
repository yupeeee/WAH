## Install

```commandline
pip install wah
```

### Requirements

You might want to manually install [**PyTorch**](https://pytorch.org/get-started/locally/)
for GPU computation.

```text
lightning
matplotlib
pandas
pyperclip
PyYAML
selenium
tensorboard
timm
torch
torchaudio
torchmetrics
torchvision
webdriver_manager
```
